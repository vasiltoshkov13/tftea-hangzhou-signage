# Hangzhou Expo Display — Run Instructions

## Current hardware reality

- No tablet catalog.
- No IWILL mini PC.
- Main display only.
- Best option: **Vasil's Mac connected to the TV/display by HDMI**.
- Backup option: **USB flash disk**, only if the TV can open/play the file format.

## Best setup — Mac + HDMI

1. Copy/open this file on the Mac:
   - `hero-display.html`
2. Open it in Chrome or Safari.
3. Connect the Mac to the booth display / TV with HDMI.
4. Set display mode to mirror or extend.
5. Put browser fullscreen:
   - Chrome/Safari fullscreen: `Control + Command + F`
   - Or presentation fullscreen if available.
6. Let the slideshow loop automatically.

## If using a USB flash disk

Put this file on the flash disk:

- `hero-display.html`

Important: many TVs **do not** play HTML files from USB. If the TV cannot open HTML, the USB fallback needs an exported **MP4 video** version of the slideshow.

If there is no Mac at the booth, ask the organizer / booth company if their screen supports:

- MP4 video loop from USB, or
- HDMI input from any laptop.
