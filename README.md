**Expo:** May 21-25, 2026 | Hangzhou International Expo Center

|--------|------|------|---------|

## Labels

5 bilingual (EN/CN) labels in `labels/` folder:
- Green Rose / 永春花园
- White Rose / 银瓣轻语
- Black Rose / 绯红丝绒
- Imperial Garden / 皇家花园
- Bulgarian Trinity / 保加利亚三宝

## Tech

- Pure HTML/CSS/JS — no server needed, runs locally on IWILL mini PCs
- Chromium kiosk mode (fullscreen, no UI chrome)
- Touch-optimized for tablets
- Auto-looping for displays
