## Main File

Open this file on the Mac:

- `hero-display.html`

It is a fullscreen looping slideshow for the booth display.

## How to Run

1. Open `hero-display.html` in Chrome or Safari.
2. Connect the Mac to the booth display / TV using HDMI or a USB-C HDMI hub.
3. Move the browser window to the display if needed.
4. Put the browser fullscreen.
5. Let the slideshow loop automatically.

Manual controls:

- Click / Space / Right Arrow = next slide
- Left Arrow = previous slide

See also:

- `DISPLAY-RUN-INSTRUCTIONS.md`

## Pricing Note

Detailed prices are intentionally **not shown on the display**.

Reason: exhibition visitors often expect negotiation. Pricing should be controlled through:

- Display Price
- Expo Special Price
- deeper negotiation only by Vasil or Iva
